## Featured Projects

### 🚀 CareerBridge — Flagship Project · Portfolio v1

A production-deployed, role-based hiring platform that supports **Candidate**, **Recruiter**, and **Admin** workflows. Built as a server-first Next.js application with authorization enforced on the server for every protected surface, validated mutations, automated testing, and a controlled deployment pipeline.

Deployed to production with a live, verified Candidate path. Recruiter, Admin, and messaging workflows are implemented and covered by automated tests; not every workflow is production-verified yet.

**Technologies**

`Next.js 16` · `React 19` · `TypeScript` · `PostgreSQL` · `Prisma` · `Better Auth` · `Tailwind CSS` · `Zod` · `React Hook Form` · `Vitest` · `GitHub Actions` · `Vercel`

**Highlights**

* Candidate, Recruiter, and Admin authorization boundaries
* Candidate profiles covering education, experience, and skills
* Recruiter companies and job management
* Job publication and moderation workflows
* Application and hiring-domain workflows
* Server-side authorization with validated mutations
* Automated unit and integration testing
* CI/CD pipeline and production deployment
* Production operations and read-only smoke-check documentation
* Private CV / document handling through a re-authorizing download route
* Localization across four locales (English, Turkish, Azerbaijani, Russian)

* Tactics editor with a recommended starting XI
* Match simulation engine with detailed post-match reports
* Context-aware press conferences and dynamic media reactions
